Blinkenworld version sqrt(2)

What is this?

Blinkenworld visualizes a dataset containing geocoordinates and timestamps on a worldmap, in this case posts from an international imageboard. You'll need a proper browser to view this (supporting native canvas and JSON).

The source code is available on GitHub under a license yet to be determined (it will be free). You are invited to fork the code, however if you make something completely different it'd be nice of you to change the name. We already got two Blinkenworlds out there.

Authors: vrs (idea, animation), erlehmann (optimizations to the HTML code)
